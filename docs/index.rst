Strata TypeScript Tutor Documentation
===========================================

**Learn TypeScript by Building Games with Strata!**

Welcome to the Strata TypeScript Tutor - an interactive, mascot-driven learning experience that teaches TypeScript through the joy of game development. Our friendly mascot Pixel guides you through creating dungeon crawlers, RPGs, platformers, racing games, and more!

.. note::

   This project is part of the `Strata <https://github.com/jbcom/strata>`_ ecosystem - a collection of tools for building amazing web experiences.

Why TypeScript?
---------------

TypeScript is the **perfect language** for learning modern programming:

- **Type Safety**: Catch errors before they happen with static typing
- **Modern JavaScript**: Learn ES6+ features with added guardrails  
- **Industry Standard**: Used by React, Angular, Vue, and game engines
- **Web Native**: Build games that run instantly in any browser
- **Great Tooling**: Excellent IDE support with autocomplete and refactoring
