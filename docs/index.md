# Strata TypeScript Tutor

> **Learn TypeScript by building amazing 3D projects with Strata**

## Welcome!

Strata TypeScript Tutor is an interactive educational platform that teaches TypeScript through hands-on projects using the [@jbcom/strata](https://www.npmjs.com/package/@jbcom/strata) procedural 3D graphics library.

## Why TypeScript?

TypeScript is JavaScript with superpowers:

- **Type Safety**: Catch bugs before they happen
- **Better Tooling**: Autocomplete, refactoring, and documentation
- **Modern Features**: Classes, async/await, and more
- **Industry Standard**: Used by companies like Microsoft, Google, and Airbnb

## What You'll Learn

### TypeScript Fundamentals
- Variables and type annotations
- Functions with typed parameters
- Interfaces and type aliases
- Classes and inheritance
- Generics for reusable code

### 3D Graphics with Strata
- Procedural terrain generation
- Water and reflection effects
- Vegetation systems
- Atmospheric effects
- Character animation

## Features

### Interactive Code Editor
Write TypeScript directly in the browser with Monaco Editor (the same editor that powers VS Code).

### Pixel the Mascot
Your friendly guide through the learning journey. Pixel provides hints, encouragement, and explanations.

### Real-Time Previews
See your 3D creations come to life instantly as you code.

### Progress Tracking
Your progress is saved automatically so you can pick up where you left off.
